# RockByte

简体中文 | [English](./README_EN.md)

为 2take1 用户开发的一个 lua 脚本，主要目标是提供快捷任务、提升公共战局体验

本项目参考了一些 [Keks-menu](https://github.com/kektram/Keks-menu) 的写法和用法。

# 快速开始

### 依赖

+ 2take1 menu

### 使用说明

1. 下载当前项目
2. 将脚本放入 %APPDATA%\PopstarDevs\2Take1Menu\scripts 目录中
3. 进入游戏加载脚本即可
